# Andrew Wulff - Portfolio Website

A professional portfolio website showcasing my projects, skills, and experience as a Computer Science student at Virginia Tech.

## About

This is my personal portfolio website built with HTML, CSS (SCSS), and JavaScript. It features a modern, responsive design with light/dark mode toggle and smooth page transitions.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Dark/Light Mode**: Toggle between dark and light themes
- **Smooth Animations**: Page transitions and hover effects for better user experience
- **Portfolio Showcase**: Display of personal projects with links to GitHub and live demos
- **Skills Section**: Visual representation of technical skills and proficiencies
- **Contact Information**: Easy access to contact details and social media links
- **Resume Download**: Direct download link for PDF resume

## Setup & Installation

1. Clone the repository:
```bash
git clone https://github.com/DroidRobot/DroidRobot.github.io.git
```

2. Navigate to the project directory:
```bash
cd DroidRobot.github.io
```

3. Open `index.html` in your browser or use a local development server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## Compilation (SCSS)

If you want to modify the styles, you'll need to compile the SCSS files:

1. Install SASS:
```bash
npm install -g sass
```

2. Watch for changes and compile automatically:
```bash
sass --watch styles/styles.scss:styles/styles.css
```
